🔌 LocalChat REST API 仕様（抜粋）

認証

POST/api/auth/register
新規ユーザー登録
パラメータ型説明
usernamestringユーザー名（2〜30文字）
passwordstringパスワード（8文字以上）
display_namestring?表示名（省略時はusername）
POST/api/auth/login
ログイン → アクセストークン取得

メッセージ

GET/api/channels/{id}/messages
チャンネルのメッセージ一覧取得（ページネーション対応）
POST/api/channels/{id}/messages
メッセージ送信（Markdown対応、返信: parent_id 指定）
PATCH/api/messages/{id}
メッセージ編集（自分のメッセージのみ）
DELETE/api/messages/{id}
メッセージ削除（論理削除）

ファイル

POST/api/files
ファイルアップロード（最大50MB、ドラッグ&ドロップ対応）
GET/api/files/{id}?token=xxx
ファイルダウンロード/インライン表示（トークン認証）

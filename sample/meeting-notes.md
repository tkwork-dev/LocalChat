### 2. 返信UIの仕様確定

Discord風の返信方式を採用:

1. メッセージホバーで「↩」ボタン表示
2. クリックで入力欄上部にバナー表示
3. 送信すると**親メッセージの直下**に表示
4. ネストの深さに応じてインデント増加（最大5段階）

```python:example.py
# 返信の深度計算（バックエンド側の参考実装）
def get_reply_depth(message, messages):
    depth = 0
    current = message
    while current.parent_id and depth < 5:
        depth += 1
        current = next((m for m in messages if m.id == current.parent_id), None)
        if not current:
            break
    return depth
```
